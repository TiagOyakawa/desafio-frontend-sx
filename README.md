## Execução

Para iniciar o projeto basta clonar o repositório [desafio-frontend-sx](https://github.com/TiagOyakawa/desafio-frontend-sx).

Em seguida instalar as dependencias:
```
npm i
```

Executar o comando:

```
npm run dev
```
